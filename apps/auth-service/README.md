# Auth Service

## Описание
Микросервис авторизации на базе NestJS и gRPC. Отвечает за:
- Регистрацию пользователей
- Аутентификацию (выдача и обновление JWT Access/Refresh токенов)
- Хеширование и верификацию паролей на базе Argon2
- Проверку валидности сессий пользователей

## Запуск
```sh
# Разработка
pnpm nx serve auth-service

# Сборка
pnpm nx build auth-service
```

## Конфигурация
Порт gRPC по умолчанию: `5002`  
Контракты обмена описаны в `auth.proto`.
Взаимодействует с базой данных `user-service`.
